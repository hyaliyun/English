为什么

我想写这篇文章的原因是，我经常从 Github 收到一个问题。实际上，我收到了很多问题，但有一个问题尤为突出：我如何才能流利地谈论许多不同的话题？

哈哈 ~

说实话，虽然我已经学了很多年英语，但我还是说不流利。

我知道你想像一个以英语为母语的人一样说话。你想听起来自然。你真的想说得流利。好吧，为了做到这一点，你必须使用我们在现实生活中实际使用的单词和表达方式。这就是我为你创建这个系列的原因：现实生活中的英语。

现在我要教你我们真正使用的单词和表达方式。你准备好了吗？

那么，让我们开始吧。

第一个话题​

我现在听到的最常见的问题之一是 我太累了。你并不孤单。在过去的一年里，我经历了人生中最大的失败。我经营的创业公司破产了，失去了财富，失去了珍贵的友情，失去了爱的人，整整一年，我睡不好觉，每天都感觉很累。

相信我，我知道那种糟糕的感觉。

所以，我们的第一个话题是：‘你为什么总是累？’

无论你是失业，还是刚分手，还是对未来感到迷茫，让我们冷静下来，像老朋友一样聊天。

在创建这门课程之前，我在网上搜索了很多关于这个话题的文章。ChatGPT 的一些 YouTube 视频和答案给了我很大的启发。我总结了几个热门视频的想法，在这里与大家分享。

除了医疗健康问题，我总结为以下三点，我们将围绕这三点进行深入讨论。

睡眠焦虑​

第一点是‘睡眠焦虑’。

睡眠焦虑是一种害怕、担心或担心睡眠的感觉，这反而使我们更难入睡。这形成了一个非常有害的恶性循环。我研究睡眠背后的科学已经有一段时间了，但即使作为一个有医学背景的人，事情也会变得非常混乱。一般来说，当我和这个领域的人交谈时，很难弄清楚哪些是好的建议可以遵循，哪些是坏的建议可以忽略。

对于睡眠焦虑，我们必须意识到以下误区：

误区一：“只要睡眠充足，什么时候睡觉都没关系”。​

20 世纪 90 年代，Russell 和他的研究小组发现，我们的眼睛里有一些细胞只能检测光线，而不是帮助我们看图像。我们眼睛里的这种光检测机制会告诉我们的大脑现在是一天中的什么时间，并帮助调节我们的生物钟。现在，由于我们的生物钟对光线敏感，这意味着我们通常希望在外面明亮时保持清醒，而在外面黑暗时保持睡眠。

事实上，过去几年，Russell 和他的研究小组一直在研究当“昼夜节律”因在奇怪的时间睡觉而受到干扰时会发生什么，例如当你有时差或上夜班时。他们发现，这些昼夜节律被干扰的人压力荷尔蒙会增加，患心脏病的风险也会增加，而且他们生病的频率也会更高。他们也更容易“情绪化”和“认知问题”。

参考视频：https://www.youtube.com/watch?v=qlf9-573MhI 00:02:28,070 - 00:02:38,210

重要的是要明白，我们体内都有一个滴答作响的生物钟。然而，我们的生物钟并不完全相同。这就是为什么你会听到人们说他们是早起的人或夜猫子。这与你的“睡眠类型”有关，它基于你在特定时间睡觉的自然倾向。科学家可以将我们分为不同的睡眠类型。

例如，一个自然早起的人发现早上是他们一天中效率最高的时间，那么他可能就是一个早晨型的人。如果你更喜欢早起，你可能会发现早上做创造性或高注意力的任务会更好。另一方面，如果你是一个夜猫子，如果你能控制你的日程安排，那也没问题。通常值得尝试调整事情，以便你在晚上做专注的工作。

误区二：每个人都需要八小时的睡眠​

事实是，人们实际需要的睡眠时间存在很大差异。因此，与其过于担心睡眠时间，并想着“哦，我昨晚只睡了七个小时，我今天会过得很糟糕”，不如做一些其他的事情来优化睡眠。

误区三：我们应该每天在同一时间起床​

通常，每天在同一时间起床是件好事，但你真的不需要太严格。另一件需要记住的事情是，我们的昼夜节律会随着我们的生命周期而改变。例如，青少年的睡眠时间更长是很常见的。夜行性睡眠型的人，这就是为什么他们早上很难醒来。

误区四：睡觉前应避免接触蓝光​

蓝光是一种波长较短、能量较大的光。科学家们推测，蓝光会以某种方式影响眼睛，使人更难入睡。

不健康的生活方式​

第二件事是“不健康的生活方式”。我的简短建议是多锻炼。白天要保持身体活跃。锻炼可以改善你的情绪、新陈代谢和能量水平。由于我对健身知之甚少，我们在这里就略过它。

应对压力​

我想分享的最重要的事情是“如何应对压力”。

管理你的压力，学会适当放松。

这里有一个视频可能对你有帮助：https://www.youtube.com/watch?v=9QiE-M1LrZk

我在知乎上的文章中提到，我有一段时间沉迷于游戏，但我用“多巴胺排毒法”克服了它。

即使你从逻辑上知道学习、锻炼、创业或其他同样有成效的事情从长远来看会给你带来更多好处，但你仍然更喜欢看电视、玩电子游戏和浏览社交媒体。

有人可能会说原因很明显。一项活动很容易，不需要太多努力，而另一项活动很难，需要你全身心投入。

但有些人似乎对定期学习、锻炼或从事他们的“副业”没有任何问题。

在我人生的低谷，有一件事一直陪伴着我前进，那就是“编程”。我之前公司的产品需要一个高性能的富文本编辑器，但是找不到符合我们需求的开源项目，所以我自己开发了一个，并在 GitHub 上开源。每当我感到沮丧和分心时，我就会默默地做这个项目。我面临和克服的一些挑战是：

  • 设计和编写编辑器
  • 开发核心和插件系统
  • 设计复杂表格的数据结构并向 slate.js 提交 PR
  • 重构渲染引擎以摆脱 React

如果你对这个项目感兴趣，你可以加入并贡献。

我很高兴经过一年多的持续开发，我终于完成了它，而不是半途而废。

今天的课程很长，但幸运的是，现在已经OVER了。

谢谢你的关注，我希望你学到了一些有用的东西。
